# Navigation Bar 导航栏

导航栏用于展示当前页面标题，并提供返回和操作入口。

---

## 组件概述

导航栏是移动应用的核心导航组件，位于页面顶部，用于展示页面标题、返回按钮和操作按钮。

---

## 组件结构

```
┌─────────────────────────────────────┐
│  ←    页面标题              ⚙️  ⋯  │
└─────────────────────────────────────┘
```

**结构组成：**
1. 左侧区域：返回按钮/关闭按钮
2. 中间区域：页面标题
3. 右侧区域：操作按钮（0-3个）

---

## 组件尺寸

| 属性 | 值 |
|------|-----|
| 高度 | 44px (iOS) / 56px (Android) |
| 背景 | `sys_color/fill/bg-secondary` (#FFFFFF) |
| 底部边框 | `1px solid sys_color/border/default` |

---

## 导航层级

### 1. 一级页面

应用的顶级页面，左侧无返回按钮，右侧可有0-3个操作图标。

```
┌─────────────────────────────────────┐
│         首页                 🔔 ⚙️  │
└─────────────────────────────────────┘
```

### 2. 二级页面

应用的子页面，左侧有返回按钮，标题居中。

```
┌─────────────────────────────────────┐
│  ←    详情页                  ⋯    │
└─────────────────────────────────────┘
```

### 3. 面板/弹窗

模态面板，左侧通常为关闭按钮（×）。

```
┌─────────────────────────────────────┐
│  ✕    筛选                   完成   │
└─────────────────────────────────────┘
```

### 4. 权限切换

需要切换身份/权限的场景，标题左侧有下拉箭头。

```
┌─────────────────────────────────────┐
│     组织名称 ▼              ⋯      │
└─────────────────────────────────────┘
```

---

## 标题样式

| 属性 | 值 |
|------|-----|
| 字号 | 16px (font/L) |
| 字重 | 500 (Medium) |
| 颜色 | `sys_color/text/primary` |
| 对齐 | 居中 |
| 最大宽度 | 页面宽度的60%，超出省略 |

---

## 图标规范

| 元素 | 尺寸 | 颜色 | 点击区域 |
|------|------|------|----------|
| 返回图标 | 24×24px | `sys_color/text/primary` 或 `sys_color/brand/primary` | 44×44px |
| 操作图标 | 24×24px | `sys_color/text/primary` | 44×44px |
| 图标间距 | 16px | - | - |

---

## 使用规范

### ✅ Do
- 标题简洁，控制在8个字符以内
- 重要操作放在右侧
- 保持导航栏高度一致

### ❌ Don't
- 不要在导航栏放置过多操作按钮
- 不要使用过长的标题
- 不要改变导航栏高度

---

## 代码示例

### CSS Variables
```css
.navbar {
  height: 44px;
  background: var(--gray-0);
  border-bottom: 1px solid var(--gray-3);
}

.navbar-title {
  font-size: 16px;
  font-weight: 500;
  color: var(--gray-10);
}

.navbar-icon {
  color: var(--brand-6);
}
```

### React 示例
```jsx
<div className="navbar">
  <div className="navbar-left">
    {showBack && <Icon name="chevron-left" />}
  </div>
  <div className="navbar-title">{title}</div>
  <div className="navbar-right">
    {actions.map(action => <Icon key={action} name={action} />)}
  </div>
</div>
```
