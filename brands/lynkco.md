## 品牌特征

- **圆角风格**: 小圆角，方正风格 (2-6px)
- **品牌色**: 蓝色系 (#4D88FB)
- **设计风格**: 简约、科技、北欧风

## 圆角变量

| 变量 | 值 | 用途 |
|------|-----|------|
| `radius/none` | 0px | 无圆角 |
| `radius/S` | 2px | 小标签 |
| `radius/M` | 4px | 大标签 |
| `radius/L` | **6px** | 输入框、卡片 |
| `radius/XL` | 8px | 浮层 |
| `radius/XXL` | **8px** | 弹窗 |
| `radius/XXXL` | **12px** | 特殊场景 |
| `radius/full` | 999px | 胶囊形状 |
| `radius/button/S` | **2px** | 小按钮（方正） |
| `radius/button/M` | **2px** | 中/大按钮（方正） |
