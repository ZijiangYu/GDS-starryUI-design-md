|------|-----|------|
| `radius/none` | 0px | 无圆角 |
| `radius/S` | 2px | 小标签 |
| `radius/M` | 4px | 大标签 |
| `radius/L` | 8px | 输入框、卡片 |
| `radius/XL` | 8px | 浮层 |
| `radius/XXL` | 12px | 弹窗 |
| `radius/XXXL` | 16px | 特殊场景 |
| `radius/full` | 999px | 胶囊形状 |
| `radius/button/S` | **999px** | 小按钮（胶囊） |
| `radius/button/M` | **999px** | 中/大按钮（胶囊） |
